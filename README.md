# Common Ground

A daily puzzle game where you sort clues into overlapping categories and uncover the common ground.

**Play now:** [thefullrob.github.io/Common-Ground](https://thefullrob.github.io/Common-Ground/)

![Common Ground social preview](./common-ground-social-preview.png)

## What is Common Ground?

**Common Ground** is a browser puzzle game built around overlapping categories.

Each puzzle gives you clues that belong in a three-circle Venn diagram. Your job is to place them in the correct regions based on how they connect. Clear **Easy** first, then unlock **Hard**.

It is designed to feel quick, satisfying, and replayable, with a clean daily format and shareable results.

## How it works

- Each day includes new puzzles
- Start with **Easy**
- Sort clues into the correct overlapping categories
- Solve it using limited tries
- Unlock **Hard** after clearing Easy
- Share your results

## Why it’s different

Unlike standard grouping games, **Common Ground** uses a **Venn diagram format**, which makes the challenge about overlap, nuance, and layered connections rather than simple one-bucket sorting.

That gives each puzzle a different feel:

- some clues belong to one category
- some belong to two
- some belong right in the center
